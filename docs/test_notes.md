# 小乐语音通话台测试文档

## 1. 测试目标

验证小乐语音通话台的核心功能是否可用，包括登录注册、用户资料、账号记忆隔离、文字问答、知识库问答、语音输入、TTS 播报、角色切换和连续通话流程。

## 8. 已知注意事项

- 语音识别依赖麦克风权限、浏览器录音能力和后端 ASR 模型。
- TTS 播报依赖后端 EdgeTTS 或浏览器语音能力。
- 大模型回复速度受 DeepSeek / OpenAI 兼容接口网络延迟影响。
- 企业知识库回答应优先引用 IT、HR、行政、财务知识库内容。
- 不同账号之间的头像、昵称、生日、聊天历史应互相隔离。
